## JavaScript Interactions

1. **Toggle Switch**
   - Clicking toggle enables/disables feature
   - Animates slider position
   - Enables/disables Configure button
   - Updates hidden form field

2. **Configure Button**
   - Click expands configuration panel below feature item
   - Click again collapses it
   - Panel animates open/close

3. **Save**
   - All settings saved as JSON in instance `settings` column
   - Validation before save (required fields when enabled)
   - Success/error notifications

## Accessibility

- Toggle switches are keyboard accessible (Tab + Space/Enter)
- ARIA labels on all interactive elements
- Focus states visible on all controls
- Color contrast meets WCAG AA
- Screen reader announces toggle state changes
